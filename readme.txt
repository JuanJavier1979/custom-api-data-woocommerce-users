== Description ==

Add a custom tab on the WooCommerce User Account screen where the user can populate some data in a textarea and get an API response.
Add a widget to show the user saved API response on the frontend.

== Installation ==

= Manual Installation =

1. Upload the entire `/custom-api-data-woocommerce-users` directory to the `/wp-content/plugins/` directory.
2. Activate `Custom Api Data Woocommerce Users` through the 'Plugins' menu in WordPress.

== Development ==
1. Clone repo
2. npm install
3. npm run dev
